What am I seeing ?

You have opened the Preview App. The Preview App connects to Online services to receive Application changes from the Application Builder in realtime.

Check you are connected to the online service by looking at the sync icon in the header panel.

sync means you are ok, sync_problem you have connection issue to the online service. Please check the browser log in this case

If sync, then go back to the Application Builder and enter some values in the Editor
